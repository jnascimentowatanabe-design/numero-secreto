🎯 Jogo do Número Secreto (JavaScript)

Um jogo simples desenvolvido em JavaScript onde o jogador deve adivinhar um número secreto gerado aleatoriamente pelo sistema.

📌 Sobre o Projeto

Este projeto foi criado para praticar:

Lógica de programação

Estruturas condicionais (if/else)

Laços de repetição

Manipulação do DOM

Geração de números aleatórios

🧠 Como Funciona

O sistema gera um número aleatório entre 1 e 100.

O jogador insere um número no campo de entrada.

O jogo informa se o número secreto é:

🔼 Maior

🔽 Menor

✅ Correto

O jogo mostra a quantidade de tentativas até o acerto.

💻 Tecnologias Utilizadas

HTML5

CSS3

JavaScript

🚀 Como Executar

Baixe os arquivos do projeto.

Abra o arquivo index.html no navegador.

Comece a jogar.
